NeuroBlocks Hybrid v2

FEATURES:
- Keyboard or Headset selection at launch
- Auto-calibration for EEG users
- Polished neon UI side panel
- Score tracking
- Stable player swapping support

INSTALL:
pip install -r requirements.txt

RUN:
python main.py

CONTROLS (Keyboard Mode):
Left/Right Arrows - Move
Up Arrow - Rotate
Down Arrow - Fast Drop

EEG Mode:
Auto-calibrates for 10 seconds.
Alpha influences movement/rotation.
Beta controls drop speed.

Works with any LSL EEG stream.
